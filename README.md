# JS Mail & Dadi

## 📌 Descrizione
Questo progetto si divide in due sfide distinte: una focalizzata sulla ricerca manuale all'interno di una lista (Mail) e l'altra sulla generazione e il confronto di numeri casuali (Gioco dei dadi).

## 🎯 Obiettivi

### 1. Controllo Mail (Logic Challenge)
L'obiettivo è verificare la presenza di un input utente all'interno di un array di email autorizzate, simulando un sistema di controllo accessi per una festa.

### 2. Gioco dei Dadi (Randomness & Logic)
Simulazione di un lancio di dadi tra un utente e il computer.

## 🛠️ Tecnologie Utilizzate
- **HTML5:** Struttura base per l'interazione.
- **JavaScript (ES6+):** Logica di generazione numeri, cicli di ricerca e controllo condizionale.
